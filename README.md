# Mood Tracker

A minimalist web app for tracking your emotional journey.

## Features

- 🎚️ Interactive slider to record mood intensity (0-5)
- 📊 Visual timeline chart with adaptive time granularity
- 📱 Mobile-friendly responsive design
- 💾 Local data storage (no server needed)
- 📤 Easy to share with others

## Usage

Visit the live app: [Your GitHub Pages URL will be here]

Simply slide to select your current mood level and click "Record now" to save your entry. The chart automatically updates to show your mood timeline.

## Tech Stack

- Pure HTML, CSS, and JavaScript
- Chart.js for data visualization
- LocalStorage for data persistence

## Development

This is a static web app that runs entirely in the browser. No build process or server required.

## Privacy

All data is stored locally in your browser. Nothing is sent to any server.
